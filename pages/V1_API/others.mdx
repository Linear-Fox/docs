# Others
Other APIs allow you to get information about the LinearFox platform and its services.

<details>
    <summary><code>GET</code> <code><b>/v1/maintenance/</b></code> <code>(Get Maintenance Status)</code></summary>
    Parameters:

    > | name       |  type                  | data type               | description                                                           |
    > |------------|------------------------|-------------------------|-----------------------------------------------------------------------|
    > | `id`       | `Path Parameter`       | `string`                | The service to check the status of.                                   |

    Responses:

    > | status code | data type     | description                                                           |
    > |-------------|---------------|-----------------------------------------------------------------------|
    > | `200`       | `object`      | The maintenance status of the service.                                |

    Example:

    Example cURL

    > ```javascript
    >  curl --location --request GET 'https://api.linearfox.com/v1/maintenance/'
    >  curl --location --request GET 'https://api.linearfox.com/v1/maintenance/1'
    >  curl --location --request GET 'https://api.linearfox.com/v1/maintenance/2'
    >  curl --location --request GET 'https://api.linearfox.com/v1/maintenance/3'
    > ```
</details>

<details>
    <summary><code>GET</code> <code><b>/v1/other/updates/</b></code> <code>(Get latest updates)</code></summary>
    Parameters:
    > | name       |  type                  | data type               | description                                                           |
    > |------------|------------------------|-------------------------|-----------------------------------------------------------------------|
    > | `limit`    | `Path Parameter`       | `integer`               | The number of updates to return.                                      |

    Responses:
    > | status code | data type     | description                                                           |
    > |-------------|---------------|-----------------------------------------------------------------------|
    > | `200`       | `object`      | The latest updates.                                                   | 

    Example:

    > ```javascript
    > curl --location --request GET 'https://api.linearfox.com/v1/other/updates/'
    > ```
</details>

<details>
    <summary><code>GET</code> <code><b>/v1/search</b></code> <code>(Search everything in LinearFox)</code></summary>
    Parameters:

    > | name       |  type                  | data type               | description                                                           |
    > |------------|------------------------|----------------|--------------------------------------------------------------|
    > | `input`    | `query`                | `string`       | The search query.                                            |

    Responses:
    > | status code | data type     | description                                                           |
    > |-------------|---------------|-----------------------------------------------------------------------|
    > | `200`       | `object`      | The search results.                                                   |
    > | `400`       | `object`      | The search query is missing.                                          |

    Example:
    > ```javascript
    > curl --location --request GET 'https://api.linearfox.com/v1/search?input=fargo'
    > ```
</details>

<details>
    <summary><code>POST</code> <code><b>/v1/other/Ping</b></code> <code>(Update user last seen)</code></summary>
    Authorization required:
        ```OAuth 2.0 Bearer Token``` or
        ```API Key```
    
    Responses:
    > | http code     | content-type                      | response   
    > |---------------|-----------------------------------|--------------------------------------------------------------------------|                                                              |
    > | `200`         | `application/json`                | `{"message": "ok",}`                                                     |
    > | `401`         | `application/json`                | `{"message": "Unauthorized & No token provided"}`                     
    
    Example:

    > ```javascript
    > curl --location --request POST 'https://api.linearfox.com/v1/other/ping' \
    >      --header 'Authorization: Bearer [token]'
    > ```
</details>
