# Users
This is an object representing a user in LinearFox. It contains public information about the user, such as their name, email address, and phone number if its public.

<details>
    <summary><code>GET</code> <code><b>$API_URL/v1/user/data/</b></code> <code>(GET profile data)</code></summary>
    Parameters:

    > | name       |  type           | data type               | description                                                           |
    > |------------|-----------------|-------------------------|-----------------------------------------------------------------------|
    > | id         |  Required Path  | string                  | User ID                                                               |

    Responses:
    > | status code | data type     | description                                                           |
    > |-------------|---------------|-----------------------------------------------------------------------|
    > | `200`       | `object`      | User data                                                             |

    Example:

    > ```javascript
    > curl --location --request GET 'https://api.linearfox.com/v1/user/data/1'
    > ```
</details>
